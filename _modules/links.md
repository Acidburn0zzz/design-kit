## Install

```bash
npm i designkit-links
```

## Usage

```html
<a href="#">This is a link</a>
```

## CSS

```css
/*
//
// Designkit-Colors
// --------------------------------------------------
*/
a {
  text-decoration: none;
  transition: color .15s ease-in;
  color: #0A83F6;
}

a:link, a:visited {
  transition: color .15s ease-in;
}

a:hover {
  transition: color .15s ease-in;
  color: #0769c6;
}

a:active {
  transition: color .15s ease-in;
}

a:focus {
  transition: color .15s ease-in;
}
```
